
# 📊 Hashtag Analyzer

A modern web application that analyzes hashtags and provides insights like popularity score, related hashtags, and trend visualization.

---

## 🚀 Features

- 🔥 **Popularity Score** — Instantly evaluates hashtag performance  
- 🔗 **Related Hashtags** — Suggests similar and trending tags  
- 📈 **Trend Visualization** — Displays weekly trend graph  
- 📌 **Trending Hashtags Board** — Shows currently trending hashtags  
- ⚡ **Fast & Interactive UI** — Smooth and responsive experience  
- 🖱️ **Clickable Hashtags** — Click any hashtag to analyze instantly  

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)  
- **Frontend:** HTML, CSS, JavaScript  
- **Visualization:** Chart.js
